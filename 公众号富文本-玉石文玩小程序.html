玉石文玩小程序 - 作品集展示

一款高端玉石文玩购物小程序，采用优雅的金色/米色配色方案，包含完整的电商功能和个人定制服务。

📱 页面展示

1. 首页

功能特点：

  • 全屏轮播图，带有"东方美玉·传承千年"主题文案
  • 特色服务展示（天然A货、匠心工艺、精美礼盒、贴心服务）
  • 产品分类横向滚动（手镯、吊坠、戒指、耳饰、摆件）
  • 平安扣与匠心工艺专题卡片
  • 品类导航（玉石类、木制类、宝石类、果核类）
  • 新品上新产品网格
  • 固定悬浮客服按钮

2. 产品列表页

功能特点：

  • 顶部搜索框
  • 分类标签切换（全部、玉石类、木制类、宝石类、果核类）
  • 产品网格布局展示
  • 底部购物车悬浮按钮
  • 价格和销量展示

3. 产品详情页

功能特点：

  • 产品图片轮播
  • 产品名称、价格、已预订数量
  • 服务详情介绍
  • 立即预订按钮
  • 底部电话咨询和立即预订双按钮
  • 右上角分享按钮
  • 固定悬浮客服按钮

4. 个人定制页

功能特点：

  • 定制主题横幅
  • 专属定制标题
  • 姓名、电话、材质、目标用途表单
  • 提交按钮
  • 固定悬浮客服按钮

5. 关于我们页

功能特点：

  • 品牌介绍标题
  • 详细文字介绍
  • 为什么选择我们（3个特色）
  • 线下门店展示卡片
  • 门店地图展示
  • 固定悬浮客服按钮

6. 我的（个人中心）

功能特点：

  • 用户头像和登录区域
  • 立即登录按钮
  • 订单状态导航（待支付、进行中、已完成、已取消）
  • 我的功能入口（最新消息、我的订阅、我的表单）
  • 底部退出登录按钮
  • 固定悬浮客服按钮

技术栈

  • 前端框架：微信小程序原生开发
  • 开发语言：WXML + WXSS + JavaScript
  • 项目类型：电商小程序

核心功能

✅ 电商功能完整 - 产品展示、分类筛选、详情查看、预订功能
✅ 个人定制服务 - 专属定制表单，支持材质和用途选择
✅ 高端视觉设计 - 金色/米色配色，优雅卡片布局
✅ 用户体验优秀 - 分类导航、轮播图、悬浮客服
✅ 响应式布局 - 完美适配各种屏幕尺寸

项目文件结构

jade-miniprogram/
├── app.js              # 小程序入口
├── app.json            # 全局配置（包含tabBar）
├── app.wxss            # 全局样式
├── project.config.json # 项目配置
├── pages/
│   ├── index/          # 首页
│   ├── product-list/   # 产品列表页
│   ├── product-detail/ # 产品详情页
│   ├── custom/         # 个人定制页
│   ├── about/          # 关于我们页
│   └── profile/        # 个人中心页
├── utils/
│   └── data.js         # 产品数据
└── images/             # tabBar图标
