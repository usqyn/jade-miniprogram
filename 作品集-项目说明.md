# 玉石文玩小程序 - 作品集展示

## 项目简介
一款高端玉石文玩购物小程序，采用优雅的金色/米色配色方案，包含完整的电商功能和个人定制服务。

**技术栈：** 微信小程序原生开发 (WXML + WXSS + JavaScript)  
**项目地址：** https://github.com/usqyn/jade-miniprogram

---

## 获取截图步骤

### 方法一：微信开发者工具（推荐）

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

2. 打开微信开发者工具，选择「导入项目」

3. 项目目录选择：`/workspace`

4. AppID 选择「测试号」或使用您的小程序 AppID

5. 点击「导入」

6. 在模拟器中依次访问各页面，使用工具栏的「截图」功能保存图片

### 方法二：真机预览（效果最佳）

1. 在微信开发者工具中点击「预览」

2. 扫码在手机微信中打开

3. 依次访问各页面，使用手机截图功能

```
│   └── profile/        # 个人中心页
├── utils/
│   └── data.js         # 产品数据

```
